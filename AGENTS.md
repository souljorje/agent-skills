# Agent Skills

Personal repo for authored agent skills and their eval briefs.

## Quick Reference

- [skills/](./skills): local skills shipped by this repo
- [evals/](./evals): test briefs for skills generated by [skill-autoresearch](https://skillsmp.com/skills/pvragon-ai-workspace-reference-team-lib-skills-skill-autoresearch-skill-md)
- [.agents/](./.agents) local agent assets, not shipped

## Rules

- Do not advertise content from `.agents/` as if it were a local published skill
- Do not edit evals/ manually outside of `skill-autoresearch` context

## Development

- On skills changes ensure running `skill-autoresearch` with min 3 briefs and update `Skills` table in [README.md](./README.md)
- Ensure each skill has `metadata` field (check format in any existing skill)
- Use conventional-commit style commit messages
