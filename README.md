# Agent Skills 🤖

A personal collection of [Agent Skills](https://agentskills.io/home) crafted by [Georgii Bukharov](https://github.com/souljorje)

## Install

```bash
npx skills add souljorje/agent-skills
```

```bash
bunx skills add souljorje/agent-skills
```

```bash
pnpm dlx skills add souljorje/agent-skills
```

## Autoresearch Evals

The `autoresearch-evals/` directory contains test briefs for skills generated by [skill-autoresearch](./skills/skill-autoresearch/SKILL.md)

They are stable inputs used to test whether a skill behaves as intended across representative scenarios and edge cases.
